# pyros

### How to test

In order to try the recommendation algorithm with the default
dataset use the following command

```sh
$ python main.py ./datasets/ml1m.tr ./datasets/ml1m.te
```

### Version
0.9.0

### Tech

Pyros requires the following python modules:

* [Scikit-learn]
* [Numpy]
* [CVXOPT]
